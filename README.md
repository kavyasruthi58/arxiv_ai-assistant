# AI Research Assistant for ArXiv Papers

This is an AI-powered research assistant that helps users search, summarize, and compare AI/ML research papers using Retrieval-Augmented Generation (RAG).

## Live App
https://arxiv-ai-assistant.streamlit.app

## Features
- Semantic search over research papers
- Grounded answers using retrieved context
- Paper summarization and comparison
- Source links with each answer

## Tech Stack
- Python
- Streamlit
- ChromaDB
- Sentence Transformers
- Groq LLaMA 3

## How to run
pip install -r requirements.txt  
streamlit run src/app.py
